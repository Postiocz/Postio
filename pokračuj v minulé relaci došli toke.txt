  2. Problém se synchronizací dat

  Podíval jsem se na kód a identifikoval jsem kritický problém:

  ❌ Hlavní problém: Dashboard je Server Component

  Tvůj page.tsx je staticky renderovaná server componenta (nemá "use client"). To znamená:

  1. Data se načtou jen jednou při buildu/server renderu
  2. Žádný automatický re-fetch po změnách (nový příspěvek, nový účet, atd.)
  3. Žádná cache invalidace – uživatel vidí stará data, dokud neprovede hard refresh

  Proč se widgety "nesynchronizují":

  1. Převést dashboard na Client Component (nebo použít hybridní přístup)
    - Přidej "use client" na začátek souboru
    - Přesun data-fetching do useEffect nebo React Query / SWR hooku
  2. Použít cache invalidaci
    - Po každé akci (vytvoření/úprava/smazání příspěvku) invalidovat dashboard cache
    - V Next.js App Router: revalidatePath('/dashboard') nebo revalidatePath('/')
